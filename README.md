# 🛒 Store Payment Analysis using Python

A beginner-friendly Python data analysis project that analyzes store payment transactions and visualizes daily payment trends using different payment methods.

This project uses Python libraries like pandas and matplotlib for data cleaning, processing, and visualization.

---

## 🚀 Features

- 📂 Load and clean CSV transaction data
- 📅 Convert and process date columns
- 💳 Calculate total payments by method
- 📊 Generate daily payment trend graphs
- 🧹 Handle missing values automatically
- 📈 Visualize Card, Cash, UPI, and Credit Note transactions

---

## 🛠 Tech Stack

- Python
- pandas
- matplotlib

---

## 📌 Payment Methods Analyzed

- Card
- Cash
- UPI
- CNote (Credit Note)

---

## 📁 Dataset

Dataset used: `Store.csv`
